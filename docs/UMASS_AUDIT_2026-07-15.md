## Summary

### Items to CHANGE (PediRef value differs from UMass)
1. Atropine RSI pretreatment → align to **0.02 mg/kg** (UMass single value)
2. Midazolam RSI → align to **0.3 mg/kg IV push** (UMass)
3. Rocuronium → align to **1 mg/kg IV** (UMass)
4. Fentanyl IV analgesia → upper bound **1 mcg/kg** (UMass narrower)
5. Midazolam IV sedation → upper bound **0.1 mg/kg** (UMass)
6. Diphenhydramine → upper bound **1.25 mg/kg** (UMass)
7. Methylprednisolone → UMass pattern: 2 mg/kg load, then 0.5 mg/kg q6h
8. Famotidine IV → **0.25 mg/kg** (UMass)
9. Ampicillin → standard **25–50 mg/kg**, meningitis 75–100 (UMass)
10. Metronidazole → **10 mg/kg** (UMass single value)
11. Ipratropium → UMass scheme: 0.5 mg neb q20min × 3
12. Magnesium (asthma) → **50 mg/kg** (UMass single value)
13. Midazolam IV seizure → **0.2 mg/kg** (UMass single value)
14. Diazepam IV seizure → upper bound **0.2 mg/kg** (UMass)
15. Fosphenytoin load → **20 mg PE/kg** (UMass single value)
16. Ondansetron → range **0.1–0.15 mg/kg** (UMass)
17. Sodium bicarbonate → add UMass age-based concentration (4.2% <2y, 8.4% ≥2y)
