# PediRef — Gaps in UMass 2026-2027 Source

These clinical content areas are **not covered** by the UMass Memorial Children's Medical Center Pediatric Emergency Medicine Medication Dosing Card 2026-2027. Each will need a separate authoritative source before the corresponding feature is fully verified.

## High-priority gaps (clinical core)

| Topic | Why UMass is silent | Candidate source(s) | Status |
|---|---|---|---|
| **Parkland burn formula** | UMass trauma section has TXA + mannitol + 3% NaCl but no burn resuscitation | AHA / ATLS pediatric guidelines; ABA (American Burn Association) | 🔴 NEEDED |
| **GCS / Pediatric GCS** | Clinical scale, not a medication | Standard GCS table (Teasdale 1974); Pediatric GCS adaptation | 🔴 NEEDED |
| **Westley Croup Score** | Clinical scale, not a medication | Original Westley 1974; AAP croup guideline | 🔴 NEEDED |
| **DKA dosing** | UMass only references "ED Pediatric DKA" Epic order set — no actual doses | ISPAD 2022 / 2024 pediatric DKA guidelines; PES pediatric DKA protocol | 🔴 NEEDED |
| **Neonatal resuscitation / dosing** | UMass explicitly: *"Please refer to Lexi-Comp or NeoFax for all Neonatal dosing"* | NeoFax; AAP NRP 2025 | 🔴 NEEDED |
| **Sepsis fluid resuscitation** | UMass has hydrocortisone stress dose only; no initial fluid bolus dosing | PALS 2025 septic shock algorithm; Surviving Sepsis Campaign pediatric | 🔴 NEEDED |
